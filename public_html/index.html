A from-scratch recreation of The Shadow of Yserbius!

Welcome!

This site is a dev blog centered around remaking The Shadow of Yserbius. I'm very close to a public release, and plan more projects in the future.

IMPORTANT! I don't have the rights to The Shadow of Yserbius. This is not an official release, just a closed-source hobby project.

Road Map

  • Feature Complete: Dungeon (January 1, 2019), Tavern (January 15, 2019).
  • Breaking bug fixes and assets expected complete February 15, 2019.
  • Additional polish and closed testing expected through mid-March 2019.
  • Public release expected mid-March 2019.
  • Hypercare and stabilization after launch expected to end May 2019.

Development Facts

This recreation runs on an engine I wrote called "CommunityKit". CommunityKit is written in C#, with the server portion running on .NET Core 2.0 (hosted in Ubuntu), and the client running on the FNA Framework in Windows and Mac.

I have been working on CommunityKit for six and a half months.

  • The CommunityKit Server/Client is 85,700 lines of code.
  • The Server-specific codebase is approximately 46% of the code.
  • The Client-specific codebase is approximately 44% of the code.
  • The Client and Server share an additional codebase of 8,500 lines of code.
